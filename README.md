# UnraidCustomScripts
Some custom scripts for Unraid server

## Prerequisites

List of plugins that need to be installed:
- Community Applications
- Dynamix S3 Sleep
- Dynamix SSD TRIM
- User Scripts

## Install

```sh
$ cd /boot/config
$ git clone https://github.com/Starli0n/UnraidCustomScripts.git custom
$ cd custom
$ bash install.sh
```
